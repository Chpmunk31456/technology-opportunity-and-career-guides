## Topics Covered

- Incident response governance
- Roles and responsibilities
- Incident classification and severity
- Detection and initial triage
- Investigation and evidence preservation
- Containment
- Eradication
- Recovery
- Communications
- Ransomware response
- Cloud and identity incidents

- Incident metrics
